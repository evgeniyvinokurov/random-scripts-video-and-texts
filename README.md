# random-scripts-video-and-texts v0.8.6  
Фабрика случайных скриптов на питоне  
  
- классы  
mixer.py - класс работы с видео для генерации коллажей  
usefull.py - класс полезных функций  
eightball.py - класс для генерации псевдослучайных значений  
randomio.py - класс работы с файловой системой для чтения байтов и текстов  
magicklib.py - класс - магия  
musicle.py - класс - музыкальные фунциии  
  
- примеры скриптов  
autoletter.py - автописьмо  
livedb.py - бредовый автомат повторного использования текста  
lottery.py - генерация чисел для лотереи  
make_playlist.py - плейлист генератор  
music_library.py - поиск неповрежденной музыки на компьютере  
play_salts_files.py - случайные файлы и картинки  
play_salts_music.py - случайная песня по картинке  
play_mixer.py - генерация видео коллажей с музыкой, настройки в файле .config  
play_mixer_salts.py - вариации выбора псевдослучайных значений и видео  
  
- скрипты Usefull  
append_audio.py - добавление музыки в видео по размеру видео  
finder.py - поиск дубликатов файлов и полное перемещение их (дублиткатов) в отдельную папку  
links.py - генерация файла sitemap.xml  
  
- js  
eightball.js - скрипт для 8шара в браузере, можно добавлять в функцию соль и получать случайное значение из массива + скрипт для моей книжной полки (+ по соли)
