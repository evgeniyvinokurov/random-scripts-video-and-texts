# random-scripts-video-and-texts v0.7.5  
Скрипты на питоне (рандом)  
  
- классы  
mixer.py - класс работы с видео для генерации коллажей  
eightball.py - класс - генерация псевдослучайных значений  
randomio.py - класс - работа с файловой системой  
magicklib.py - класс - магия (пока автописьмо)  
usefull.py - класс - полезное, то что подойдет для бытовых нужд  
  
- скрипты (в будущем только примеры, библиотеки см. выше)  
playmixer.py - генерация видео коллажей с музыкой, настройки в файле .config  
randomMixerSalts.py - вариации выбора псевдослучайных значений и видео  
playlistmaker.py - плейлист генератор  
musicLibrary.py - поиск неповрежденной музыки на компьютере  
playautoletter.py - автописьмо  
lottery.py - генерация чисел для лотереи  
livedb.py - бредовый автомат повторного использования текста  
playsalts.py - случайная песня по картинке  
play_salts_files.py - случайные файлы и картинки  
  
- скрипты Usefull  
finder.py - поиск дубликатов файлов и полное перемещение их (дублиткатов) в отдельную папку  
appendAudio.py - добавление музыки в видео п оразмеру видео  
links.py - генерация файла sitemap.xml  
  
- js  
eightball.js - скрипт для 8шара в браузере, можно добавлять в функцию соль и получать случайное значение из массива + скрипт для моей книжной полки (+ по соли)
